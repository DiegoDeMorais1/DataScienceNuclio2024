Training course notebooks, class 2024

M1 - Data Science Tools - Module dedicated to developing the foundations of the Python and SQL languages. About the Python language, how to define variables, lists, dictionaries, as well as the use of conditions, iterators, functions.
* Python - Variables, operators, lists and dictionaries
* Python - Numpy and Pandas Series
* Python - Pandas DataFrame + good practices using Pandas
* SQL - Select + Joins + Good practices in SQL

M2 - Data Analytics - Module focused on data analysis, understanding how to analyze it in detail and, whenever necessary, clean or transform it. Data visualization and exploratory analysis.
* Perform data transformations, pre-processing, standardizations and visualizations
* Data Visualization in Python

M3 - Machine Learning - Supervised and unsupervised learning.
* Supervised learning: Classification and regression
* Unsupervised Learning: Clustering and Collaborative Filtering

M4 - Deep Learning
* Neural network and its components
* Different neural network architectures
* Process unstructured information

M5 - MLOps
* MLOps tools
* Introduction to the Cloud: Docker, Kubernetes and AirFlow
* Put a Data Science model into production
